# Smart Student Record & Performance Manager

A console-based Student Management System built using C++.  
This project helps manage student records with subject-wise marks, attendance tracking, grade calculation and performance analysis.

## Features

- Add Student Record
- Display Students
- Search Student
- Update Student Information
- Delete Student Record
- Subject-wise Marks Tracking
- Automatic Percentage Calculation
- Grade Generation
- Attendance Monitoring
- Low Attendance Detection
- Class Topper Detection
- Class Average Calculation

## Technologies Used

- C++
- Structures
- Vectors
- Functions
- Menu Driven Programming
