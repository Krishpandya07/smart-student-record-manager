# Smart Student Record Manager

A console-based Student Management System built using C++.

## Features
- Add Student
- Display Students
- Search Student
- Update Student
- Delete Student
- Subject-wise Marks
- Percentage Calculation
- Grade Generation
- Attendance Tracking
- Low Attendance Detection
- Class Topper Detection
- Class Average Calculation

## Technologies Used
- C++
- Structures
- Vectors
- Functions
- Menu Driven Programming

## How to Run

Compile the program

g++ main.cpp -o student

Run the program

./student
